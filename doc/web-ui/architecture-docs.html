System Architecture Overview

The VSA Control Plane is a microservices-based system built with Go, using Temporal for workflow orchestration and PostgreSQL for data persistence. The architecture follows clean architecture principles with clear separation of concerns.

Core Service

Main business logic and orchestration

Google Proxy

GCP API integration and routing

Database

PostgreSQL for data persistence

Temporal

Workflow orchestration engine

Telemetry

Metrics and monitoring

Worker

Temporal workflow execution
